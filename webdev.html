My latest work

First Portfolio

Created a website from scratch using lessons learned during my on-the-job training, applying this knowledge to build a fully responsive portfolio inspired by professional designs, including Brittany Chiang's. Using HTML, CSS, and Vanilla JavaScript. The layout dynamically adjusts, maintaining usability on both desktop and mobile platforms. I emphasized clean aesthetics and intuitive navigation, highlighting my front-end development and UI/UX design skills.

Gadget Store

Created a dynamic, full-stack website for ordering gadgets online. The platform features user authentication with Clerk and integrates Stripe for secure payments. It has also a responsive shopping cart that enhances user experience and implemented webhook connectivity for real-time updates to the admin panel. The MongoDB database efficiently manages data of products, categories, orders, and customers, ensuring seamless communication between the front and admin interfaces.

Food Delivery App

Developed a full stack food delivery website featuring user authentication, database management, and a seamless food ordering experience. The site allows users to create accounts, log in, and manage their shopping cart for food items. It integrates Stripe for secure payments and provides real-time order status updates. The project was built using React.js for the front end, Node.js and Express for the backend, and MongoDB for database management, with development done in VS Code.

Ecommerce Admin

Built a full stack ecommerce admin panel with API connectivity to the front end, enabling efficient management of products, categories, and users. The panel allows for product creation, editing, and deletion, along with user management using Clerk for secure Google and Facebook authentication. The project was developed using Next.js for the framework, styled with Tailwind CSS, and utilized MongoDB for the database. Development was carried out in WebStorm for streamlined coding.

Food Menu

Designed a dynamic food menu application using React JS and Canva that allows users to browse and order from a wide selection of dishes. The app features an interactive interface with categories, item descriptions, and images, ensuring a user-friendly experience. Each menu item includes details such as price, ingredients, and availability, and users can easily add items to their cart for checkout. The menu dynamically updates based on user selections, offering a seamless browsing experience.

POS Inventory

Developed a Point of Sale (POS) management inventory system during my second year of college to help a local company manage sales and inventory. The system includes user login functionality and a comprehensive database that tracks products, customers, categories, users, and orders. Each item in the database is assigned a unique ID for easy management, and the system automatically generates detailed receipts for each transaction. The project was built using VB.net, Microsoft SQL Server, and Visual Studio IDE.
